My personal portfolio/blog.  The website is a generated static website.  It's built with Flask, Frozen-Flask, Skeleton.css.

"python build.py" builds the site into the deploy directory using markdown blog posts from the pages directory.

I took inspiration for the site from https://github.com/n1k0/nicolas.perriault.net, https://github.com/florapdx/My-Blog and https://github.com/gatesphere/ptah .
